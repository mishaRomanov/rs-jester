# rs-jester - a very simple reverse proxy based on [pingora](https://github.com/cloudflare/pingora), the cloudflare's rust framework. 

# Usage 
`git clone git@github.com:mishaRomanov/rs-jester.git`

`make help`

## Useful stuff for local development
`bexmpl/` contains a very simple `Go` backend to demonstrate the reverse proxy functionality.
